# 股票年度已实现盈利计算器

为了应对CRS监管，合理报税，本项目利用量化API自动获取各个平台的股票交易记录并计算每个年度已实现盈利，作为报税依据。

## 支持平台

| 平台 | 状态 |
|------|------|
| 富途牛牛 | ✅ 已支持 |
| 长桥 | ✅ 已支持 |

## 功能特性

- 自动获取交易记录和现金流水
- 支持多种成本核算方法
- 按年度统计已实现收益
- 支持期权交易和过期处理
- Web可视化界面

## 成本核算方法

本项目支持以下成本核算方法：

| 方法 | 说明 | 税务适用性 |
|------|------|-----------|
| **移动加权平均法** | 将所有买入成本加权平均 | 默认方法，国际通用 |
| **FIFO** | 先进先出法，先买入的先卖出 | 美国IRS默认方法 |
| **LIFO** | 后进先出法，后买入的先卖出 | 仅作参考，美国不允许 |
| **HIFO** | 最高成本优先法 | 税务筹划参考 |
| **LOFO** | 最低成本优先法 | 税务筹划参考 |

### 各国税法要求

| 地区 | 推荐方法 |
|------|---------|
| 美国税务居民 | FIFO 或 Specific Identification |
| CRS报税 | 移动加权平均法 |

## 成本核算算法说明

### 移动加权平均法（默认）

将所有买入成本加权平均，计算公式：

```
真实买入价 = 买入价格 + 费用/数量
真实卖出价 = 卖出价格 - 费用/数量
持仓均价 = 总成本 / 总数量
已实现收益 = 平仓数量 × (真实卖出价 - 持仓均价)
```

### FIFO（先进先出法）

假设先买入的股票先卖出，按买入顺序确定成本。

### LIFO（后进先出法）

假设后买入的股票先卖出，按买入逆序确定成本。

### HIFO（最高成本优先法）

假设成本最高的股票先卖出，可最大化成本、最小化利润。

### LOFO（最低成本优先法）

假设成本最低的股票先卖出，最小化成本、最大化利润。

## 注意事项

1. **税务合规**：不同国家/地区对成本核算方法有不同要求，请根据您的税务居民身份选择合适的方法
2. **数据安全**：API密钥请妥善保管，不要提交到版本控制系统
3. **期权处理**：系统会自动检测并处理过期期权
4. **做空支持**：支持做空交易的收益计算
